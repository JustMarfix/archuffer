class BitWriter:

View In Hierarchy

Bit-packing writer.

Accumulates individual bits into bytes and buffers them until flushed.

Method __init__ Initialize an empty bit writer.
Method flush Flush remaining bits (if any) and return the full byte buffer.
Method write_bits Write the lowest nbits of value to the buffer, MSB first.
Method write_bytes Write raw bytes, aligning pending bits to the next byte boundary.
Instance Variable bit_buffer 8-bit scratch register for accumulating pending bits.
Instance Variable bit_count Number of valid bits currently stored in bit_buffer (0-7).
Instance Variable buffer Internal byte buffer holding fully written bytes.
