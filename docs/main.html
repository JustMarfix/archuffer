def create_archive(targets: list[str], output_path: str, hide_progress: bool): ¶

Create an archive file from given targets (files and/or directories).

Archive format (little-endian): - Magic: 'ARH1' (4 bytes) - Version: 1 byte - Entry count: uint32 For each entry: - Path length: uint32 - Path (utf-8 bytes) - Type: uint8 (0=file, 1=dir) - Metadata (since container VERSION >= 2): - - mode: uint32 (POSIX permission bits, see stat.S_IMODE) - - uid: uint32 (0xFFFFFFFF if unknown) - - gid: uint32 (0xFFFFFFFF if unknown) - If file: - - Compressed size: uint32 - - Compressed data bytes (produced by Archiver.compress)
Parameters
targets:List[str]Filesystem targets to include (each file/dir is added recursively).
output_path:strDestination archive file path.
hide_progress:boolWhether to show per-file and overall progress.
Returns
NoneNone
